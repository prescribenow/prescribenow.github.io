Edit User

At times, there is a need to modify a User. Opting for 'Edit User' permits you to implement these modifications. Upon clicking 'Edit User,' your profile is presented, allowing for changes to be made. Certain text fields are confined from modification due to software limitations. A stop sign appears in the restricted text field.

A provider may select more than one facility at a time.

Ensure that your ‘User Status’ is set to Active. If the user status is marked as inactive:
• no invitation email will be sent
• the provider’s information will be stored in the system
• the provider will not have the ability to log into the system.

By now, name of the provider has been added into the system, but status of the provider is set to ‘Pending’. To change status to ‘Active’, the provider should access and reply to the sent email.

Click on the link given in the email.

You will be directed to a screen where you will add and reconfirm your website password. Click on ‘Submit’ to activate your account.

Your status will change from ‘Pending’ to ‘Active’.
