Prescription Tab

The Prescription tab allows healthcare providers to create, manage, and update prescriptions for patients. It includes a comprehensive record of a patient's medication history, including dosage instructions and refill requests. A Prescription sample screenshot is shown below.

At the end of each prescription there are two symbols. The cursive arrow allows you to ‘Reorder’ a certain medicine for the mentioned number of days. Clicking on it opens a new medication window titled ‘Reorder Prescription’.

Refer to Medication Order for details of these fields.

Another symbol is a cross; when chosen, it cancels the prescription. Prior to cancellation, a new window emerges, providing detailed information about the prescription. When ‘Cancel Rx’ is clicked, the status of the prescription changes to ‘Cancellation Pending’.
