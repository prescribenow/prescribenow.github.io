User Roles

In a medical software, User Roles are like labels that show what each person using the system can do. These labels decide who can see what information and what they're allowed to do in the software.

User Roles can be accessed by selecting ‘Admin’ from the ‘Home’ screen, and then clicking on ‘User Roles’.

User Roles is divided into eight categories: They are:
1. Patients
2. Encounters
3. Appointments
4. Tasks
5. Account Settings
6. Messages
7. Eligibility Checks / Claims / Patient Invoices
8. Practice Management

Of course, this can be changed by editing the ‘User Role’ of Practice Admin and Billing Specialist. To do this, click on ‘Edit User Roles’ at the upper right corner of the screen. Next, check the boxes under Practice Admin and Billing Specialist for the said category. When the changes have been saved, Practice Admin and Billing Specialist will now be able to view patient medication orders.

There are some check boxes that cannot be unchecked. For example, a Physician’s right to view a patient’s medication order is mandatory. Therefore, by default, this permission cannot be undone. When the main category is selected, all associated subcategories will also be selected.
